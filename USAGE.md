# 电商图片抓取器 - 使用说明

## 🎯 功能概述

这是一个专为电商网站设计的浏览器扩展，可以智能识别商品详情页中的图片，并提供批量下载和分类管理功能。

## ✨ 主要功能

### 🖼️ 智能图片抓取
- 自动识别电商网站商品详情页的所有图片
- 智能分类：商品主图、详情图、买家评价、缩略图等
- 支持多种主流电商平台

### 🎨 现代化界面
- 右侧浮动工具栏，不影响页面浏览
- 美观的图片展示面板
- 响应式设计，支持网格和列表视图
- 现代化的动画效果

### 📥 批量操作
- 单张图片下载
- 批量选择和下载
- 复制图片链接到剪贴板
- 搜索和筛选功能

## 🛒 支持的网站

- 淘宝/天猫 (taobao.com, tmall.com)
- 京东 (jd.com)
- Amazon (amazon.com, amazon.cn)
- 苏宁易购 (suning.com)
- 拼多多 (pinduoduo.com)
- 唯品会 (vip.com)

## 🚀 使用方法

### 1. 安装扩展
1. 构建项目：`pnpm run build`
2. 打开Chrome浏览器，进入 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目的 `.output/chrome-mv3` 文件夹

### 2. 使用扩展
1. 访问任意支持的商品详情页
2. 页面右侧会自动出现浮动工具栏
3. 点击"抓取图片"按钮
4. 在右侧弹窗中查看和管理图片
5. 使用搜索、筛选和批量下载功能

### 3. 功能说明

#### 工具栏按钮
- **抓取图片**：开始识别页面中的图片
- **图片计数**：显示已抓取的图片数量

#### 图片管理面板
- **搜索框**：按图片名称或链接搜索
- **分类筛选**：按图片类型筛选（主图、详情图、评价图等）
- **视图切换**：网格视图和列表视图
- **批量操作**：全选、批量下载、复制链接
- **单张操作**：单独下载每张图片

## 🔧 开发说明

### 技术栈
- **WXT**: 现代浏览器扩展开发框架
- **Svelte 5**: 前端框架
- **Tailwind CSS**: 样式框架
- **Lucide Svelte**: 图标库
- **TypeScript**: 类型安全

### 项目结构
```
src/
├── components/          # Svelte组件
│   ├── Toolbar.svelte  # 右侧浮动工具栏
│   └── ImagePanel.svelte # 图片展示面板
├── entrypoints/        # 扩展入口点
│   ├── content.ts      # 内容脚本
│   ├── background.ts   # 后台脚本
│   └── popup/          # 弹窗界面
└── styles/            # 全局样式
    └── global.css      # Tailwind CSS和自定义样式
```

### 开发命令
```bash
# 开发模式
pnpm run dev

# 构建项目
pnpm run build

# 打包扩展
pnpm run zip

# 类型检查
pnpm run check
```

## 📝 注意事项

1. **权限要求**：扩展需要访问页面内容和下载权限
2. **网络限制**：某些网站可能有反爬虫机制
3. **图片质量**：抓取的图片质量取决于网站显示的图片
4. **浏览器兼容性**：主要针对Chrome/Edge浏览器优化

## 🐛 问题反馈

如果遇到问题，请检查：
1. 是否在支持的商品详情页
2. 浏览器是否已启用扩展
3. 网络连接是否正常
4. 页面是否完全加载
